プライバシーポリシー

Math Keyboard / 最終更新日：2026年1月4日

「Math Keyboard」（以下「本アプリ」）は、iOSのカスタムキーボード（Keyboard Extension）として動作する数式入力用キーボードです。 本ポリシーでは、本アプリにおけるユーザー情報の取り扱いについて説明します。

1. 収集する情報について

本アプリは、ユーザーの入力内容（キーボードで入力した文字、文章、数式など）を 収集・保存・送信しません。

また、本アプリは以下の情報を収集しません。

  • 氏名、メールアドレス等の個人情報
  • 位置情報
  • 連絡先
  • 写真・メディア
  • 広告ID等の識別子
  • 利用ログ（分析目的のトラッキング等）

2. フルアクセスについて

iOSの仕様上、キーボード拡張で一部機能（例：ホストアプリを開く、アプリグループで設定を共有する等）を利用するには 「フルアクセス」を許可する必要があります。

本アプリは「フルアクセス」がONの場合でも、入力内容を収集・外部送信することはありません。

3. 保存する情報（端末内）

本アプリは、キーボードの動作に必要な最小限の設定情報（キーボード設定、課金状況、表示設定など）を、 端末内（またはApp Group領域）に保存する場合があります。

これらの情報はユーザーの端末内にのみ保存され、外部に送信されません。

4. 第三者提供について

本アプリは、収集した情報を第三者に提供しません。 そもそも本アプリは入力内容や個人情報を収集しないため、提供の対象となる情報がありません。

5. 外部送信・通信について

本アプリは、通常のキーボード入力機能のために外部サーバーへ通信を行いません。

6. Pro（買い切り）について

本アプリには、Pro（買い切り）によって解放される機能があります。購入処理はAppleの仕組み（App Store / StoreKit）を用いて行われます。

本アプリ側でクレジットカード情報などを保持することはありません。 提供されるPro機能の内容は、アプリ内の表示・説明に従います。
